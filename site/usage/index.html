    User Guide¶

    Installation¶

    git clone https://github.com/yourorg/foremost
cd foremost
source .venv/bin/activate
pip install -e .[dev,docs]

    Mode 0 — Synthetic landscape¶

    The fastest way to verify the installation and explore the full pipeline. No input files required.

    # CLI
foremost --mode 0

# With Hydra overrides
foremost optimizer.algo=NSGA2 optimizer.pop_size=100 optimizer.n_gen=50

    from foremost import run_demo, ForemostConfig

cfg = ForemostConfig()
cfg.data.nrows = 20
cfg.data.ncols = 20
cfg.optimizer.algo = "NSGA2"
cfg.optimizer.n_gen = 50

run_demo(mode=0, cfg=cfg)

    Mode 1 — Load .npy arrays from disk¶

    Use pre-exported NumPy arrays from the annotator (or any external source).

    foremost --mode 1 --npy-folder outputs/

    from foremost import load_npy_arrays, ForemostProblemBuilder
from foremost.core import _arrays_to_habitatdata

arrays = load_npy_arrays("outputs/")
data = _arrays_to_habitatdata(arrays, cell_area=1.0)

result = (
    ForemostProblemBuilder(data)
    .set_full_objective()
    .add_restorable_constraint(min_restore=20.0, max_restore=200.0)
    .add_compactness_constraint(max_diameter=9)
    .add_connected_constraint(max_nb_cc=3)
    .add_budget_constraint(max_cost=20000.0)
    .solve(pop_size=80, n_gen=120, algo="NSGA2")
)

    Mode 2 — Interactive annotation then optimize¶

    Launch the annotation GUI, label the landscape, then run the optimizer on the resulting arrays.

    foremost --mode 2 --image zone.tif
foremost --mode 2 --gpkg zones.gpkg

    Configuration with Hydra¶

    Generate the default config files:

    foremost --write-config    # creates conf/foremost.yaml
annotate --write-config    # creates conf/annotator.yaml

    Then override any parameter from the CLI:

    foremost optimizer.algo=CTAEA optimizer.pop_size=150 optimizer.n_gen=200
foremost data.nrows=50 data.ncols=50 constraints.max_diameter=12

    Running tests¶

    # All tests
pytest tests/ -v

# With coverage report
pytest tests/ --cov=foremost --cov-report=term-missing

# Fast tests only (exclude solver runs)
pytest tests/ -v -m "not timeout"

    Building docs¶

    # Live development server
mkdocs serve

# Production build
mkdocs build --strict
